# BKK Election 65 → 69 — เปรียบเทียบผลการเลือกตั้ง ส.ก.

Visualization เปรียบเทียบผู้ชนะ (ผู้สมัครอันดับ 1) ของสมาชิกสภากรุงเทพมหานคร (ส.ก.) ทั้ง 50 เขต ระหว่างปี 2565 และปี 2569 แยกเป็น 2 มุมมอง

- **① ที่นั่งเปลี่ยนมือ (มองที่ "เขต")** — แต่ละเขต พรรค/กลุ่มใดชนะปี 65 และปี 69 หน่วยคือ *เขต* (เส้นไม่ได้แปลว่าคนเดิมย้ายพรรค)
- **② คนเดิมย้ายพรรค (มองที่ "คน")** — เฉพาะผู้ชนะคนเดียวกันที่ลงทั้งสองปี และย้ายพรรค/กลุ่มจริง หน่วยคือ *คน*

คลิกที่เส้นเพื่อดูรายละเอียดเทียบรายเขต/รายคน (พรรค · ชื่อ · คะแนน) และมีปุ่มสลับ **โหมดสว่าง/มืด** มุมขวาบน

หมายเหตุเชิงตรรกะ: พรรค **ก้าวไกล (65)** และ **ประชาชน (69)** ถือเป็นพรรคเดียวกัน (ยุบแล้วตั้งใหม่) จึงไม่นับเป็นการย้ายพรรค

## โครงสร้างไฟล์

- **`data.js`** — ⭐ ข้อมูลทั้งหมด (ผลแต่ละเขต ปี 65/69, สีพรรค, % นับคะแนน, วันที่อัปเดต) — **แก้ที่ไฟล์นี้ไฟล์เดียว** เวลาข้อมูลเปลี่ยน
- `index.html` — logic + การวาดกราฟ (คำนวณโหนด/เส้น/คนเดิม จาก `data.js` ทั้งหมด ไม่มีค่า hard-code)
- `*.otf` — ฟอนต์ SaoChingcha / BKKDraft5
- `.github/workflows/deploy.yml` — ดีพลอยขึ้น GitHub Pages อัตโนมัติเมื่อ push เข้า `main`

## แก้ไขข้อมูล

เปิด `data.js` แล้วแก้ได้เลย — เช่น
- เปลี่ยนคะแนน/ผู้ชนะของเขตใด: แก้ใน `year65` / `year69`
- เพิ่มพรรคใหม่: เพิ่มใน `parties` (กำหนดสี + `canon`); `canon` เดียวกัน = ถือเป็นพรรคเดียวกัน (เช่น ก้าวไกล=ประชาชน, เพื่อไทย=เพื่อไทย Life)
- ซ่อนเส้นในกราฟ② (พรรคเดิม/เปลี่ยนชื่อล้วน): เพิ่มคู่ใน `hidePersonFlows`
- ปรับ % นับคะแนน หรือ วันที่อัปเดต: แก้ใน `meta`

กราฟทั้งหมดจะคำนวณใหม่เองจากข้อมูล ไม่ต้องแตะ `index.html`

## Local

เปิด `index.html` ด้วยเบราว์เซอร์ได้เลย (ไม่ต้องมี build step)
